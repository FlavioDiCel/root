Contrast, Repetition, Alignment, and Proximity

Contrast

Banco de Chile
  • The website uses strong color contrast with blue, white, and red to highlight important actions like buttons.
  • Headings are larger and bolder than body text, creating a clear visual hierarchy.
  • Clickable elements stand out due to their distinct colors compared to the rest of the page.

Repetition

ALLMusic
  1. The website uses repeated layouts across pages, including a consistent navigation bar and content structure.
  2. Content is displayed using repeated card designs for albums and artists.
  3. The same color scheme (mainly blue and white) is used throughout the site.
  4. Interactive elements like ratings and buttons are consistent across pages.

Alignment

EarthWin
Centered Alignment
The website uses centered alignment for headings, text, and buttons, creating a balanced and visually appealing layout.
Vertical Structure
Content is organized in vertically stacked sections, which improves readability and creates a clear visual flow.
Form Alignment
Form elements such as input fields are aligned vertically, making them easy to follow and interact with.
Consistent Layout
Related elements are aligned consistently across the page, helping users scan and understand the content quickly.

Proximity

NBA
  • The website groups related content into clear sections such as news, scores, and videos, improving organization.
  • Content is displayed in cards where images, titles, and descriptions are placed close together to show they belong to the same item.
  • Game information such as teams, scores, and schedules is grouped in single visual blocks, making it easy to understand.
  • White space is used effectively to separate different sections and avoid visual clutter.
  • Navigation menu items are grouped together logically, helping users find related content quickly.
